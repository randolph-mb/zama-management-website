# Image Placeholders

This directory contains placeholder images for the Zama Management website.

## Required Images:

1. **logo_zama.png** - Main header logo (recommended: 120x40px)
2. **logo_zama_footer.png** - Footer logo (recommended: 120x40px)
3. **zama_kontakt.png** - Contact page hero image (recommended: 1920x1080px)

## How to Replace Placeholders:

1. Add your actual images to this `/public/assets/img/` directory
2. Ensure the filenames match exactly as listed above
3. Optimize images for web before uploading (compress, appropriate format)
